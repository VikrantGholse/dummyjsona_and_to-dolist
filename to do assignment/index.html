ABOUT

The website is a straightforward landing page with a to-do list feature and a picture gallery. Users can browse between the Home, Gallery, and To-Do List sections using the navigation bar at the top of the page. While the Gallery area shows a dynamically filled image gallery, the Home section greets visitors to the main page. Users can add tasks to the To-Do List section, where they appear as list items. With a backdrop image and an adaptable layout that works on a range of devices, the page has a simple, minimalistic design.
